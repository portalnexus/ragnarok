Bem-vindo à TUI/CLI Wiki

Seu guia completo para construir belas Text User Interfaces e Command Line Interfaces em múltiplas linguagens de programação.

Começar a Aprender → Navegar Frameworks
demo.py 
from textual.app import App, ComposeResult
from textual.widgets import Header, Footer, Static

class OlaMundo(App):
    def compose(self) -> ComposeResult:
        yield Header()
        yield Static("Olá, Mundo!")
        yield Footer()

if __name__ == "__main__":
    app = OlaMundo()
    app.run()

Por que TUI/CLI?

🚀

Ultra Rápido

Sem overhead de navegador. Execute diretamente no seu terminal com inicialização instantânea.

🔧

Pronto para Remoto

SSH em qualquer servidor e tenha uma interface interativa completa na ponta dos dedos.

♿

Acessível

Design focado em teclado torna TUIs naturalmente acessíveis para todos os usuários.

📦

Leve

Dependências mínimas e bundles pequenos, perfeito para qualquer ambiente.
