Welcome to TUI/CLI Wiki

Your comprehensive guide to building beautiful Text User Interfaces and Command Line Interfaces across multiple programming languages.

Start Learning → Browse Frameworks
demo.py 
from textual.app import App, ComposeResult
from textual.widgets import Header, Footer, Static

class HelloWorld(App):
    def compose(self) -> ComposeResult:
        yield Header()
        yield Static("Hello, World!")
        yield Footer()

if __name__ == "__main__":
    app = HelloWorld()
    app.run()

Why TUI/CLI?

🚀

Lightning Fast

No browser overhead. Run directly in your terminal with instant startup times.

🔧

Remote Ready

SSH into any server and have a full interactive interface at your fingertips.

♿

Accessible

Keyboard-first design makes TUIs naturally accessible for all users.

📦

Lightweight

Minimal dependencies and small bundle sizes perfect for any environment.
