Python Basics for TUI

Learn Python fundamentals for building terminal applications

📋 Prerequisites

Before diving into TUI development, you should understand these Python basics:

  • Variables and data types
  • Functions and parameters
  • Lists and dictionaries
  • Loops (for/while)
  • Conditional statements (if/elif/else)
  • Import statements

🐍 Hello World

Your first Python program:

print("Hello, World!")

Run it:

python hello.py

📦 Variables and Data Types

# Strings (text)
name = "Alice"
message = 'Hello, ' + name

# Integers (whole numbers)
age = 25
count = 0

# Floats (decimal numbers)
price = 19.99
temperature = -5.5

# Booleans (True/False)
is_active = True
has_permission = False

# Print values
print(name)        # Alice
print(age)         # 25
print(is_active)   # True

📝 Lists (Arrays)

Lists store multiple items in a single variable:

# Create a list
fruits = ["apple", "banana", "cherry"]

# Access items (0-indexed)
print(fruits[0])  # apple
print(fruits[1])  # banana

# Add items
fruits.append("orange")

# Remove items
fruits.remove("banana")

# Loop through list
for fruit in fruits:
    print(fruit)

# List length
print(len(fruits))  # 3

🔑 Dictionaries (Key-Value Pairs)

Dictionaries store data as key-value pairs:

# Create a dictionary
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Access values
print(person["name"])  # Alice
print(person["age"])   # 25

# Add/Update values
person["age"] = 26
person["email"] = "alice@example.com"

# Loop through dictionary
for key, value in person.items():
    print(f"{key}: {value}")

# Check if key exists
if "name" in person:
    print("Name exists!")

🔁 Loops

For Loop

# Loop through a range
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Loop with index
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

While Loop

count = 0
while count < 5:
    print(count)
    count += 1  # Increment count

⚡ Conditional Statements

age = 18

if age < 13:
    print("Child")
elif age < 20:
    print("Teenager")
else:
    print("Adult")

# Multiple conditions
score = 85
if score >= 90:
    grade = "A"
elif score >= 80 and score < 90:  # 'and' operator
    grade = "B"
elif score >= 70 or score < 80:   # 'or' operator
    grade = "C"
else:
    grade = "F"

🔧 Functions

Functions are reusable blocks of code:

# Define a function
def greet(name):
    return f"Hello, {name}!"

# Call the function
message = greet("Alice")
print(message)  # Hello, Alice!

# Function with default parameter
def greet_person(name="Guest"):
    return f"Welcome, {name}!"

print(greet_person())        # Welcome, Guest!
print(greet_person("Bob"))   # Welcome, Bob!

# Function with multiple parameters
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8

📥 Import Statements

Import modules to use their functions:

# Import entire module
import math
print(math.sqrt(16))  # 4.0

# Import specific function
from random import randint
print(randint(1, 10))  # Random number 1-10

# Import with alias
import datetime as dt
print(dt.datetime.now())

# Import all from module (not recommended)
from math import *
print(pi)  # 3.14159...

🎯 Practice Exercise

Create a simple calculator:

def calculator(a, b, operation):
    """Simple calculator function"""
    if operation == "+":
        return a + b
    elif operation == "-":
        return a - b
    elif operation == "*":
        return a * b
    elif operation == "/":
        if b != 0:
            return a / b
        else:
            return "Error: Division by zero"
    else:
        return "Invalid operation"

# Test the calculator
print(calculator(10, 5, "+"))  # 15
print(calculator(10, 5, "-"))  # 10
print(calculator(10, 5, "*"))  # 50
print(calculator(10, 5, "/"))  # 2.0
