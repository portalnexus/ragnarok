Python Básico para TUI

Aprenda os fundamentos de Python para construir aplicações de terminal

📋 Pré-requisitos

Antes de mergulhar no desenvolvimento TUI, você deve entender estes conceitos básicos de Python:

  • Variáveis e tipos de dados
  • Funções e parâmetros
  • Listas e dicionários
  • Loops (for/while)
  • Estruturas condicionais (if/elif/else)
  • Importação de módulos

🐍 Olá, Mundo!

Seu primeiro programa em Python:

print("Olá, Mundo!")

Execute:

python hello.py

📦 Variáveis e Tipos de Dados

# Strings (texto)
nome = "Alice"
mensagem = 'Olá, ' + nome

# Inteiros (números inteiros)
idade = 25
contador = 0

# Floats (números decimais)
preco = 19.99
temperatura = -5.5

# Booleans (Verdadeiro/Falso)
ativo = True
tem_permissao = False

# Imprimir valores
print(nome)        # Alice
print(idade)       # 25
print(ativo)       # True

📝 Listas (Arrays)

Listas armazenam múltiplos itens em uma única variável:

# Criar uma lista
frutas = ["maçã", "banana", "cereja"]

# Acessar itens (índice 0)
print(frutas[0])  # maçã
print(frutas[1])  # banana

# Adicionar itens
frutas.append("laranja")

# Remover itens
frutas.remove("banana")

# Percorrer lista
for fruta in frutas:
    print(fruta)

# Tamanho da lista
print(len(frutas))  # 3

🔑 Dicionários (Pares Chave-Valor)

Dicionários armazenam dados como pares chave-valor:

# Criar um dicionário
pessoa = {
    "nome": "Alice",
    "idade": 25,
    "cidade": "Nova York"
}

# Acessar valores
print(pessoa["nome"])   # Alice
print(pessoa["idade"])  # 25

# Adicionar/Atualizar valores
pessoa["idade"] = 26
pessoa["email"] = "alice@exemplo.com"

# Percorrer dicionário
for chave, valor in pessoa.items():
    print(f"{chave}: {valor}")

# Verificar se chave existe
if "nome" in pessoa:
    print("Nome existe!")

🔁 Loops

Loop For

# Loop através de um range
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# Loop através de uma lista
frutas = ["maçã", "banana", "cereja"]
for fruta in frutas:
    print(fruta)

# Loop com índice
for i, fruta in enumerate(frutas):
    print(f"{i}: {fruta}")

Loop While

contador = 0
while contador < 5:
    print(contador)
    contador += 1  # Incrementar contador

⚡ Estruturas Condicionais

idade = 18

if idade < 13:
    print("Criança")
elif idade < 20:
    print("Adolescente")
else:
    print("Adulto")

# Múltiplas condições
nota = 85
if nota >= 90:
    conceito = "A"
elif nota >= 80 and nota < 90:  # operador 'and' (e)
    conceito = "B"
elif nota >= 70 or nota < 80:   # operador 'or' (ou)
    conceito = "C"
else:
    conceito = "F"

🔧 Funções

Funções são blocos de código reutilizáveis:

# Definir uma função
def saudar(nome):
    return f"Olá, {nome}!"

# Chamar a função
mensagem = saudar("Alice")
print(mensagem)  # Olá, Alice!

# Função com parâmetro padrão
def saudar_pessoa(nome="Visitante"):
    return f"Bem-vindo, {nome}!"

print(saudar_pessoa())         # Bem-vindo, Visitante!
print(saudar_pessoa("Bob"))    # Bem-vindo, Bob!

# Função com múltiplos parâmetros
def somar(a, b):
    return a + b

resultado = somar(5, 3)
print(resultado)  # 8

📥 Importação de Módulos

Importe módulos para usar suas funções:

# Importar módulo inteiro
import math
print(math.sqrt(16))  # 4.0

# Importar função específica
from random import randint
print(randint(1, 10))  # Número aleatório 1-10

# Importar com apelido (alias)
import datetime as dt
print(dt.datetime.now())

# Importar tudo do módulo (não recomendado)
from math import *
print(pi)  # 3.14159...

🎯 Exercício Prático

Crie uma calculadora simples:

def calculadora(a, b, operacao):
    """Função calculadora simples"""
    if operacao == "+":
        return a + b
    elif operacao == "-":
        return a - b
    elif operacao == "*":
        return a * b
    elif operacao == "/":
        if b != 0:
            return a / b
        else:
            return "Erro: Divisão por zero"
    else:
        return "Operação inválida"

# Testar a calculadora
print(calculadora(10, 5, "+"))  # 15
print(calculadora(10, 5, "-"))  # 10
print(calculadora(10, 5, "*"))  # 50
print(calculadora(10, 5, "/"))  # 2.0
