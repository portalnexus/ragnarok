Implementing Themes

Python (Rich)

from rich.style import Style
from rich.color import Color

# Define theme colors
THEME = {
    "background": Color.parse("#282a36"),
    "foreground": Color.parse("#f8f8f2"),
    "accent": Color.parse("#bd93f9"),
    "success": Color.parse("#50fa7b"),
    "error": Color.parse("#ff5555"),
}

# Apply to text
style = Style(color=THEME["foreground"], 
              bgcolor=THEME["background"])
print(style.render("Themed text"))

Rust (Ratatui)

use ratatui::style::{Color, Style};

// Define theme
struct Theme {
    bg: Color,
    fg: Color,
    accent: Color,
}

impl Theme {
    fn dracula() -> Self {
        Self {
            bg: Color::Rgb(40, 42, 54),
            fg: Color::Rgb(248, 248, 242),
            accent: Color::Rgb(189, 147, 249),
        }
    }
}

Best Practices

🎨 Respect User Settings

Detect and use the terminal's configured colors when possible.

🔆 Ensure Contrast

Maintain at least 4.5:1 contrast ratio for readability.

🌈 Provide Options

Offer light and dark theme variants.

⚠️ Graceful Degradation

Fall back gracefully on terminals with limited color support.

Color Blind Considerations

Approximately 8% of men have color vision deficiency:

  • Avoid: Red/Green combinations
  • Use: Blue/Orange or Blue/Red
  • Add: Icons, patterns, or text labels
