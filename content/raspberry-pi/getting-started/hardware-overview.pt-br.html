Visão Geral do Hardware Raspberry Pi 3B

Guia completo dos componentes físicos e layout do Pi 3B

📋 Layout da Placa

O Raspberry Pi 3B é um computador de placa única compacto medindo 85,6 × 56,5 mm. Entender o layout é essencial para configuração adequada e planejamento de projetos.

Áreas Principais:
  • Borda Superior: Cabeçalho GPIO (40 pinos)
  • Lado Direito: Portas USB e Ethernet
  • Borda Inferior: HDMI, áudio/composto, energia MicroUSB
  • Lado Esquerdo: Slot para cartão MicroSD
  • Centro: SoC (System on Chip) com RAM empilhada

🔌 Portas e Conectores

Energia (MicroUSB)

Localização: Borda inferior, perto do jack de áudio
Especificação: 5V/2.5A recomendado
Notas: Use uma fonte de qualidade. Energia insuficiente causa instabilidade.

Portas USB 2.0 (4x)

Localização: Lado direito, empilhadas
Especificação: USB 2.0, até 480 Mbps
Uso: Teclado, mouse, adaptador WiFi, armazenamento externo

⚠️ Limitação de Energia: Corrente USB total limitada a 1.2A. Para dispositivos de alta potência, use um hub USB alimentado.

Ethernet (RJ45)

Localização: Lado direito, abaixo das portas USB
Especificação: 10/100 Mbps (Fast Ethernet)
Notas: Barramento USB compartilhado limita throughput a ~300 Mbps máx

HDMI (Tamanho Completo)

Localização: Borda inferior
Especificação: HDMI 1.4, até 1080p60
Uso: Saída de display primária

Áudio/Composto (Jack 3.5mm)

Localização: Borda inferior, próximo ao HDMI
Especificação: Áudio estéreo analógico + vídeo composto
Notas: Requer cabo especial com conector TRRS

Slot para Cartão MicroSD

Localização: Borda inferior (parte inferior)
Especificação: Suporta até 256GB+ (testado)
Notas: Empurre para inserir/ejetar. O cartão clica no lugar.

⚠️ Segurança GPIO:
  • Pinos GPIO são lógica 3.3V - NÃO toleram 5V!
  • Corrente máxima por pino: 16mA
  • Corrente total de todos os pinos: máx 50mA
  • Sempre use resistores limitadores de corrente com LEDs
  • Use level shifters para dispositivos 5V

💾 Opções de Armazenamento

Cartão MicroSD (Primário)

# Especificações recomendadas:
- Capacidade: 32GB mínimo, 64GB+ recomendado
- Velocidade: Class 10 / UHS-I
- Formato: FAT32 (para partição de boot)
- Marcas: SanDisk, Samsung, Kingston

Boot USB (Alternativo)

O Pi 3B pode inicializar de armazenamento USB com bootloader atualizado: 

# Habilitar boot USB
echo program_usb_boot_mode=1 | sudo tee -a /boot/config.txt
sudo reboot
vcgencmd otp_dump | grep 17:  # Deve retornar 17:3020000a

🌡️ Energia e Térmica

Consumo de Energia

Estado Corrente Potência
Ocioso (desktop)~350mA~1.75W
Ocioso (CLI)~250mA~1.25W
Carga (4 núcleos)~600mA~3W
Máx (com periféricos)~2.5A~12.5W

Considerações Térmicas

O Pi 3B pode sofrer throttling em temperaturas acima de 85°C. Para desenvolvimento TUI/CLI:

  • Ocioso: Sem dissipador necessário para trabalho CLI básico
  • Carga sustentada: Dissipadores recomendados
  • Projetos em gabinete: Resfriamento ativo (ventoinha) aconselhado

🔧 Acessórios Recomendados

Essenciais

✓ Fonte de alimentação 5V/2.5A de qualidade (com MicroUSB)
✓ Cartão MicroSD 32GB+ (Class 10)
✓ Cabo HDMI
✓ Teclado e mouse USB
✓ Gabinete com ventilação

Para Projetos GPIO

✓ Protoboard (400+ pontos)
✓ Jumpers (M-M, M-F, F-F)
✓ LEDs (cores variadas)
✓ Resistores (220Ω, 330Ω, 1kΩ, 10kΩ)
✓ Botões push-button
✓ Placa de expansão GPIO

Opcionais mas Úteis

✓ Adaptador WiFi USB (se não usar o integrado)
✓ Adaptador Ethernet USB (para Gigabit via hub USB 3.0)
✓ Dissipadores ou ventoinha
✓ Hub USB (alimentado, para múltiplos periféricos)
✓ Módulo RTC (para cronometragem sem rede)
