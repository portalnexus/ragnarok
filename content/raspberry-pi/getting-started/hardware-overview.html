Raspberry Pi 3B Hardware Overview

Complete guide to the physical components and layout of the Pi 3B

📋 Board Layout

The Raspberry Pi 3B is a compact single-board computer measuring 85.6 × 56.5 mm. Understanding the layout is essential for proper setup and project planning.

Key Areas:
  • Top Edge: GPIO header (40 pins)
  • Right Side: USB ports and Ethernet
  • Bottom Edge: HDMI, audio/composite, MicroUSB power
  • Left Side: MicroSD card slot
  • Center: SoC (System on Chip) with RAM stacked

🔌 Ports and Connectors

Power (MicroUSB)

Location: Bottom edge, near audio jack
Specification: 5V/2.5A recommended
Notes: Use a quality power supply. Insufficient power causes instability.

USB 2.0 Ports (4x)

Location: Right side, stacked
Specification: USB 2.0, up to 480 Mbps
Usage: Keyboard, mouse, WiFi adapter, external storage

⚠️ Power Limitation: Total USB current limited to 1.2A. For high-power devices, use a powered USB hub.

Ethernet (RJ45)

Location: Right side, below USB ports
Specification: 10/100 Mbps (Fast Ethernet)
Notes: Shared USB bus limits throughput to ~300 Mbps max

HDMI (Full Size)

Location: Bottom edge
Specification: HDMI 1.4, up to 1080p60
Usage: Primary display output

Audio/Composite (3.5mm Jack)

Location: Bottom edge, next to HDMI
Specification: Analog stereo audio + composite video
Notes: Requires special cable with TRRS connector

MicroSD Card Slot

Location: Bottom edge (underside)
Specification: Supports up to 256GB+ (tested)
Notes: Push to insert/eject. Card clicks into place.

⚠️ GPIO Safety:
  • GPIO pins are 3.3V logic - NOT 5V tolerant!
  • Maximum current per pin: 16mA
  • Total current from all pins: 50mA max
  • Always use current-limiting resistors with LEDs
  • Use level shifters for 5V devices

💾 Storage Options

MicroSD Card (Primary)

# Recommended specifications:
- Capacity: 32GB minimum, 64GB+ recommended
- Speed Class: Class 10 / UHS-I
- Format: FAT32 (for boot partition)
- Brands: SanDisk, Samsung, Kingston

USB Boot (Alternative)

Pi 3B can boot from USB storage with updated bootloader: 

# Enable USB boot
echo program_usb_boot_mode=1 | sudo tee -a /boot/config.txt
sudo reboot
vcgencmd otp_dump | grep 17:  # Should return 17:3020000a

🌡️ Power and Thermal

Power Consumption

State Current Power
Idle (desktop)~350mA~1.75W
Idle (CLI)~250mA~1.25W
Load (4 cores)~600mA~3W
Max (with peripherals)~2.5A~12.5W

Thermal Considerations

The Pi 3B can throttle at temperatures above 85°C. For TUI/CLI development:

  • Idle: No heatsink needed for basic CLI work
  • Sustained load: Heatsinks recommended
  • Enclosed projects: Active cooling (fan) advised

🔧 Recommended Accessories

Essential

✓ Quality 5V/2.5A power supply (with MicroUSB)
✓ MicroSD card 32GB+ (Class 10)
✓ HDMI cable
✓ USB keyboard and mouse
✓ Case with ventilation

For GPIO Projects

✓ Breadboard (400+ tie points)
✓ Jumper wires (M-M, M-F, F-F)
✓ LEDs (assorted colors)
✓ Resistors (220Ω, 330Ω, 1kΩ, 10kΩ)
✓ Push buttons
✓ GPIO breakout board

Optional but Useful

✓ USB WiFi adapter (if not using built-in)
✓ USB Ethernet adapter (for Gigabit via USB 3.0 hub)
✓ Heatsinks or fan
✓ USB hub (powered, for multiple peripherals)
✓ RTC module (for timekeeping without network)
