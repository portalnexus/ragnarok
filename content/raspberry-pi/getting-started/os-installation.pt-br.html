Raspberry Pi OS Installation

Step-by-step guide to installing the operating system

📋 What You'll Need

Requirements:
  • Raspberry Pi 3B board
  • MicroSD card (16GB minimum, 32GB+ recommended)
  • Computer with SD card reader (or USB adapter)
  • Internet connection
  • 5-10 minutes of time

📥 Method 1: Raspberry Pi Imager (Recommended)

Step 1: Download Raspberry Pi Imager

Available for Windows, macOS, and Linux:

# Download from:
# https://www.raspberrypi.com/software/

Step 2: Choose OS

For TUI/CLI development, we recommend:

  • Raspberry Pi OS Lite (32-bit) - No desktop, CLI only (recommended)
  • Raspberry Pi OS (32-bit) - With desktop environment

Step 3: Choose Storage

Select your MicroSD card from the list.

Step 4: Configure (Optional)

Click the gear icon to pre-configure:

  • Hostname
  • SSH access
  • WiFi credentials
  • User account

Step 5: Write

Click "Write" and wait for completion (~5-10 minutes).

📥 Method 2: Manual Installation

Step 1: Download OS Image

# Download from:
# https://www.raspberrypi.com/software/operating-systems/

# Or use wget (Linux):
wget https://downloads.raspberrypi.com/raspios_lite_armhf_latest.zip

Step 2: Extract Image

unzip raspios_lite_armhf_latest.zip
# Results in: *.img file

Step 3: Identify SD Card

# Linux:
lsblk

# macOS:
diskutil list

# Windows:
diskpart
list disk

Step 4: Write Image

# Linux/macOS (replace /dev/sdX with your device):
sudo dd if=raspios_lite_armhf.img of=/dev/sdX bs=4M status=progress
sync

# Windows: Use Rufus or Win32 Disk Imager
⚠️ Warning: Double-check the device name! Writing to the wrong device will erase data.

⚙️ First-Boot Configuration

Before first boot, you can configure settings:

Enable SSH

# Create empty 'ssh' file in boot partition:
touch /Volumes/boot/ssh

# Or on Linux:
touch /media/boot/ssh

Configure WiFi

# Create wpa_supplicant.conf in boot partition:
country=BR
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
    ssid="YOUR_WIFI_NAME"
    psk="YOUR_PASSWORD"
    key_mgmt=WPA-PSK
}

✅ Verify Installation

After writing:

  1. Safely eject the MicroSD card
  2. Insert into Raspberry Pi
  3. Connect power
  4. Check for activity LED
  5. Wait 2-3 minutes for first boot
