Introdução ao Raspberry Pi 3B

Seu ponto de partida para aprender desenvolvimento TUI/CLI em um computador de placa única acessível

🎯 O que é Raspberry Pi?

O Raspberry Pi é um computador de placa única do tamanho de um cartão de crédito, projetado para promover o ensino de ciência da computação e programação. O Raspberry Pi 3B, lançado em 2016, permanece como uma excelente plataforma para aprender desenvolvimento TUI/CLI devido a:

  • Acessibilidade: Cerca de $35 USD (aproximadamente R$ 180-250)
  • Baixo consumo: Perfeito para operação 24/7
  • Baseado em Linux: Acesso completo ao terminal
  • Pinos GPIO: Capacidades de interação com hardware
  • Comunidade ativa: Suporte extensivo e recursos abundantes

📋 Especificações do Raspberry Pi 3B

Componente Especificação
CPU Broadcom BCM2837, Quad-core Cortex-A53 @ 1.2GHz
GPU Broadcom VideoCore IV @ 400MHz
RAM 1GB LPDDR2
Armazenamento Slot para cartão MicroSD (até 256GB+)
USB 4x portas USB 2.0
Rede Ethernet 10/100, Wi-Fi 802.11n, Bluetooth 4.1
GPIO Cabeçalho de 40 pinos
Vídeo HDMI, Composto (via conector 3.5mm)
Energia 5V/2.5A via MicroUSB

💡 Por que Usar Pi 3B para Aprender TUI/CLI?

🖥️ Experiência Linux Real

Sistema Linux completo baseado em Debian com acesso total ao terminal. Perfeito para aprender comandos Unix e shell scripting.

🔌 Interação com Hardware

40 pinos GPIO permitem controlar LEDs, ler sensores e construir projetos físicos controlados por suas aplicações TUI.

💰 Econômico

Por ~$35, é acessível para estudantes e hobbistas. Sem risco de danificar uma workstation cara.

🔋 Baixo Consumo

Consome apenas 2.5-7W, tornando-o perfeito para projetos sempre ligados como dashboards, monitores e automação residencial.

📚 Documentação Extensiva

Milhões de usuários, inúmeros tutoriais e uma comunidade solidária tornam a resolução de problemas mais fácil.

🚀 Projetos Portáteis

Tamanho pequeno permite incorporar projetos em gabinetes, criando dispositivos independentes com interfaces TUI.

🛒 O Que Você Vai Precisar

Itens Essenciais

✓ Placa Raspberry Pi 3B (~$35 / R$ 180-250)
✓ Cartão MicroSD (16GB mínimo, 32GB+ recomendado) (~$10-20 / R$ 50-100)
✓ Fonte de alimentação: 5V/2.5A com conector MicroUSB (~$10 / R$ 50-80)
✓ Cabo HDMI e monitor/TV (~$10-50)
✓ Teclado e mouse USB (~$20-40 / R$ 100-200)
✓ Leitor de cartão SD (se seu computador não tiver um) (~$10 / R$ 50)

Acessórios Recomendados

✓ Gabinete para proteção (~$10-20 / R$ 50-100)
✓ Dissipadores de calor ou ventilador para resfriamento (~$5-10 / R$ 25-50)
✓ Cabo Ethernet para rede cabeada (~$5 / R$ 25-40)
✓ Kit iniciante GPIO (LEDs, botões, resistores, jumpers) (~$15-25 / R$ 75-125)
✓ Protoboard para prototipar circuitos (~$5-10 / R$ 25-50)
💡 Dica: Muitos varejistas oferecem kits iniciantes completos que incluem a maioria destes itens com preço descontado. Procure por "Kit Iniciante Raspberry Pi 3B".

📊 Pi 3B vs Outros Modelos

Recurso Pi 3B Pi 4B Pi 5 Pi Zero 2W
CPU Quad 1.2GHz Quad 1.5GHz Quad 2.4GHz Quad 1.2GHz
RAM 1GB 2/4/8GB 4/8GB 512MB
USB 4x USB 2.0 2x USB 3.0, 2x USB 2.0 2x USB 3.0, 2x USB 2.0 1x Micro USB
Ethernet 10/100 Gigabit Gigabit Nenhuma (apenas Wi-Fi)
Preço $35 $35-75 $60-80 $15
Ideal Para Aprender TUI/CLI Substituto de desktop Alta performance Projetos compactos

Por que Pi 3B? Para desenvolvimento TUI/CLI, o Pi 3B oferece o equilíbrio perfeito entre desempenho e preço. Aplicações de terminal não precisam do poder extra do Pi 4/5, e as portas de tamanho completo facilitam o trabalho comparado à série Zero.
