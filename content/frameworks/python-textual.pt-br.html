Python Textual

Construa belas aplicações TUI com Python

Visão Geral

Textual é um framework Python moderno para construir aplicações de terminal ricas. Criado pelos desenvolvedores do Rich, usa uma arquitetura async-first e estilização tipo CSS para widgets.

Principais Recursos:
  • Suporte async/await para UIs responsivas
  • Stylesheet tipo CSS para estilizar widgets
  • Widgets built-in (Button, Input, DataTable, etc.)
  • Cross-platform (Linux, macOS, Windows)
  • Suporte a mouse

Instalação

pip install textual

Olá Mundo

from textual.app import App, ComposeResult
from textual.widgets import Header, Footer, Static

class OlaMundo(App):
    def compose(self) -> ComposeResult:
        yield Header()
        yield Static("Olá, Mundo!")
        yield Footer()

if __name__ == "__main__":
    app = OlaMundo()
    app.run()

Estilização CSS

Textual usa sintaxe tipo CSS para estilização:

/* app.tcss */
Screen {
    background: #1a1b26;
}

Header {
    background: #7aa2f7;
    color: #1a1b26;
}

Button {
    background: #bb9af7;
    color: #1a1b26;
}

Button:hover {
    background: #9d7cd8;
}

Manipulação de Eventos

from textual.app import App
from textual.widgets import Button
from textual.events import Key

class MyApp(App):
    def compose(self):
        yield Button("Clique em mim!", id="meu-botao")
    
    def on_button_pressed(self, event: Button.Pressed) -> None:
        self.notify("Botão foi pressionado!")
    
    def on_key(self, event: Key) -> None:
        if event.key == "q":
            self.exit()
