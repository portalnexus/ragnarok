Python Rich

Saída de terminal bonita em Python

Visão Geral

Rich é uma biblioteca Python para escrever texto rico (com cor e estilo) no terminal. Pode exibir tabelas bonitas, código com syntax highlighting, barras de progresso e muito mais.

Instalação

pip install rich

Uso Básico

from rich import print
from rich.text import Text

# Saída colorida
print("[bold red]Olá[/bold red] [green]Mundo![/green]")

# Usando objeto Text
texto = Text("Texto estilizado", style="bold blue on yellow")
print(texto)

Tabelas

from rich.table import Table
from rich.console import Console

console = Console()
tabela = Table(title="Filmes Star Wars")

tabela.add_column("Lançamento", style="cyan")
tabela.add_column("Título", style="magenta")
tabela.add_column("Bilheteria", justify="right", style="green")

tabela.add_row("20 Dez, 2019", "Star Wars: Ascensão Skywalker", "$952,110,690")
tabela.add_row("25 Mai, 2018", "Solo: Uma História Star Wars", "$393,151,347")

console.print(tabela)

Syntax Highlighting

from rich.console import Console
from rich.syntax import Syntax

console = Console()
codigo = '''
def ola_mundo():
    print("Olá, Mundo!")
'''
syntax = Syntax(codigo, "python", theme="monokai", line_numbers=True)
console.print(syntax)

Barras de Progresso

from rich.progress import Progress
import time

with Progress() as progress:
    tarefa = progress.add_task("Carregando...", total=100)
    while not progress.finished:
        progress.update(tarefa, advance=1)
        time.sleep(0.02)

Painéis e Layouts

from rich.panel import Panel
from rich.layout import Layout

layout = Layout()
layout.split(
    Layout(name="cabecalho", size=3),
    Layout(name="corpo"),
    Layout(name="rodape", size=3)
)

layout["corpo"].update(Panel("Conteúdo aqui"))
console.print(layout)
