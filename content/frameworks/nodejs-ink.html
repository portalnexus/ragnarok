Node.js Ink

Build CLI interfaces with React

Overview

Ink is a React renderer for building interactive terminal interfaces. If you know React, you already know how to build TUIs with Ink. It's used by popular tools like Vercel's CLI and GitHub's Copilot CLI.

Installation

npm init -y
npm install ink react

Hello World

import {render, Text} from 'ink';

const App = () => (
    <Text color="green">Hello, World!
);

render(<App />);

Core Components

ComponentDescription
<Text>Styled text with colors
<Box>Flexbox container
<Static>Output that doesn't re-render
<Newline>Line break
<Spacer>Flexible space
<Transform>Transform children output

Interactive Example

import {render, Text, useInput} from 'ink';
import {useState, useEffect} from 'react';

const Counter = () => {
    const [count, setCount] = useState(0);
    
    useInput((input, key) => {
        if (key.upArrow) setCount(c => c + 1);
        if (key.downArrow) setCount(c => c - 1);
        if (input === 'q') process.exit();
    });
    
    return (
        <Box flexDirection="column">
            <Text>Count: {count}</Text>
            <Text color="gray">Use ↑↓ to change, q to quit</Text>
        </Box>
    );
};

render(<Counter />);

Hooks

import {useInput, useApp, useStdin} from 'ink';

// Handle keyboard input
useInput((input, key) => {
    if (key.ctrl && input === 'c') exit();
});

// Access app methods
const {exit} = useApp();

// Access stdin
const {stdin, setRawMode} = useStdin();
