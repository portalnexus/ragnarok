Node.js Ink

Construa interfaces CLI com React

Visão Geral

Ink é um renderizador React para construir interfaces de terminal interativas. Se você sabe React, já sabe construir TUIs com Ink. É usado por ferramentas populares como CLI da Vercel e Copilot CLI do GitHub.

Instalação

npm init -y
npm install ink react

Olá Mundo

import {render, Text} from 'ink';

const App = () => (
    <Text color="green">Olá, Mundo!</Text>
);

render(<App />);

Exemplo Interativo

import {render, Text, useInput} from 'ink';
import {useState, useEffect} from 'react';

const Contador = () => {
    const [count, setCount] = useState(0);
    
    useInput((input, key) => {
        if (key.upArrow) setCount(c => c + 1);
        if (key.downArrow) setCount(c => c - 1);
        if (input === 'q') process.exit();
    });
    
    return (
        <Box flexDirection="column">
            <Text>Contador: {count}</Text>
            <Text color="gray">Use ↑↓ para mudar, q para sair</Text>
        </Box>
    );
};

render(<Contador />);

Hooks

import {useInput, useApp, useStdin} from 'ink';

// Manipular input de teclado
useInput((input, key) => {
    if (key.ctrl && input === 'c') exit();
});

// Acessar métodos do app
const {exit} = useApp();

// Acessar stdin
const {stdin, setRawMode} = useStdin();
