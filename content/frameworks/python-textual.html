Python Textual

Build beautiful TUI applications with Python

Overview

Textual is a modern Python framework for building rich terminal applications. Created by the developers of Rich, it uses an async-first architecture and CSS-like styling for widgets.

Key Features:
  • Async/await support for responsive UIs
  • CSS-like stylesheet for styling widgets
  • Built-in widgets (Button, Input, DataTable, etc.)
  • Cross-platform (Linux, macOS, Windows)
  • Mouse support

Installation

pip install textual

Hello World

from textual.app import App, ComposeResult
from textual.widgets import Header, Footer, Static

class HelloWorld(App):
    def compose(self) -> ComposeResult:
        yield Header()
        yield Static("Hello, World!")
        yield Footer()

if __name__ == "__main__":
    app = HelloWorld()
    app.run()

CSS Styling

Textual uses CSS-like syntax for styling:

/* app.tcss */
Screen {
    background: #1a1b26;
}

Header {
    background: #7aa2f7;
    color: #1a1b26;
}

Button {
    background: #bb9af7;
    color: #1a1b26;
}

Button:hover {
    background: #9d7cd8;
}

Event Handling

from textual.app import App
from textual.widgets import Button
from textual.events import Key

class MyApp(App):
    def compose(self):
        yield Button("Click me!", id="my-button")
    
    def on_button_pressed(self, event: Button.Pressed) -> None:
        self.notify("Button was pressed!")
    
    def on_key(self, event: Key) -> None:
        if event.key == "q":
            self.exit()
